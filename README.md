Object Related Scene View Shortcuts:

| **Command** | **Shortcut** |
| :------- | :------- |
| Focus on an object | F |
| Duplicate an object | ctrl + D |
| Rotate around focal object | Alt + Right click |
| Zoom in/out from focal object | Alt + Left click |

General Scene View Shortcuts

| **Command** | **Shortcut** |
| :------- | :------- |
| Control the scene camera | Right click + WASD |
| Zoom in/out | Scroll wheel |
| Pan | Hold scroll wheel |
